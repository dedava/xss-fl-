# xss-fl-

## Flask гостевая книга (XSS-safe)

Небольшое веб‑приложение на Flask: гостевая книга с хранением записей в SQLite и базовой защитой от XSS.

### Как запустить (Windows / PowerShell)

Создайте виртуальное окружение и установите зависимости:

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Запустите сервер:

```bash
python app.py
```

Откройте в браузере `http://127.0.0.1:5000`.

### Где здесь защита от XSS

- **Автоэкранирование Jinja2**: в шаблонах вывод `{{ ... }}` по умолчанию HTML-экранируется.
- **Серверная очистка ввода (Bleach)**: перед сохранением удаляются любые HTML‑теги/атрибуты (приложение хранит только текст).
- **CSP и заголовки безопасности**: выставляются `Content-Security-Policy`, `X-Frame-Options`, `nosniff` и др.
- **CSRF защита формы**: через Flask‑WTF (`{{ form.csrf_token }}`).

### Быстрый тест

Введите сообщение:

`<script>alert(1)</script><b>hello</b>`

Ожидаемо: в гостевой книге сохранится/отобразится только безопасный текст без исполнения скриптов и без HTML‑форматирования.
